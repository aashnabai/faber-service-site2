How to publish on GitHub Pages:
1. Create a repository on GitHub.
2. Upload these files to the repository.
3. In Settings > Pages, choose Branch: main and Folder: root, then Save.
4. Wait a minute and open https://YOUR_USERNAME.github.io/REPO_NAME/

How to publish on Netlify:
1. Extract the folder and drag the folder (not the zip) to Netlify Drop or upload the folder via 'Deploy site'.
2. Or connect your GitHub repo in Netlify and it will deploy automatically.

Form setup:
- The booking form currently posts to a placeholder Formspree endpoint.
- To receive form submissions by email, create a Formspree form (https://formspree.io) and replace the form 'action' attribute in index.html with your endpoint.
